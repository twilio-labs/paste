## Guidelines

### About Table

A Table is used to represent tabular data, or information presented in a 2-dimensional table composed of rows and columns of cells containing data.
Tables fill 100% width of the space of their containing element.

You can create tables using a combination of:

- **Table:** The table wrapper.
- **THead:** The header of your table.
- **TBody:** The main body of your table.
- **TFoot:** The footer of your table.
- **Tr:** A row of cells in your table.
- **Th:** A header cell which can be used as a column title.
- **Td:** A cell of table content.

### Accessibility

Tables are used to organize data with a logical relationship in grids. Accessible tables need HTML markup that indicates
header cells and data cells and defines their relationship. Assistive technologies use this information to provide context to users.

In order to be fully accessible:

- Tables **must** have a `THead`, containing a Table Header `Th` for each column.
- Tables **must** have a `TBody` wrapping the table body of rows.

## Usage Guide

### API

#### Installation

```bash
yarn add @twilio-paste/table - or - yarn add @twilio-paste/core
```

#### Usage

```jsx
import {Table} from ‘@twilio-paste/table’;

const TableExample: React.FC = () => {
  return (
    <Table tableLayout="fixed">
      <THead>
        <Tr verticalAlign="middle">
          <Th width="size40" textAlign="left">Column 1</Th>
          <Th>Column 2</Th>
        </Tr>
      </THead>
      <TBody>
        <Tr>
          <Td>Content</Td>
          <Td>Content</Td>
        </Tr>
      </TBody>
      <TFoot>
        <Tr>
          <Td>Content</Td>
          <Td>Content</Td>
        </Tr>
      </TFoot>
    </Table>
  );
};
```

#### Props

#### Table

##### `tableLayout?: “auto” | “fixed”`

Sets the `table-layout` style of the Table. Defaults to `auto`.

#### Tr Props

##### `verticalAlign?: “top” | “middle” | “bottom”`

Sets the vertical alignment of the content within the Table row. Defaults to `middle`.

#### Th Props

##### `textAlign?: “left” | “center” | “right”`

Sets the text alignment of the content within the Table cell. Defaults to `left`.

##### `width?: WidthOptions`

Sets the width of a Table cell.

#### Td Props

##### `textAlign?: “left” | “center” | “right”`

Sets the text alignment of the content within the Table cell. Defaults to `left`.
