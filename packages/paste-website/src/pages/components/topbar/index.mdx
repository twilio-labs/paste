## Guidelines

### Accessibility

- Set a value for `topbarSkipLinkID` to be used with Sidebar skip link functionality. The ID must be a unique string not used elsewhere in the application.

### About Topbar

A Topbar is sticky to the top of the screen and consists of application-wide controls including:

Account-specific controls (aligned to the left) such as:

- [Account Switcher](/components/account-switcher)
- Account type indicators, such as subaccount [Badges](/components/badge)
- Possible upgrade actions

User-specific controls (aligned to the right) such as:

- Search [Input](/components/input)
- Support [Button](/components/button)
- [Product Switcher](/components/product-switcher)
- [User Dialog](/components/user-dialog)

Each set of controls should be contained within a `TopbarActions` component, which comes with a `justify` property. You can justify these controls to either the `start` or the `end` of the `Topbar`.

## Usage guide

### Figma

The Topbar component is available in the Paste Components library as a [Navigation component](https://www.figma.com/file/l2qtpQVMmXzQPVB3rEAA8i/%5BNEW%5D-Paste-Components?type=design&node-id=27837-187276&t=xcPacI1AN13MooNC-0).

### API

#### Installation

```bash
yarn add @twilio-paste/topbar - or - yarn add @twilio-paste/core
```

#### Usage

```jsx
import {Topbar} from '@twilio-paste/core/topbar';

const Topbar = () => {
  return (
    <Topbar topbarSkipLinkID="topbar">
      <TopbarActions justify="start">{actions}</TopbarActions>
      <TopbarActions justify="end">{actions}</TopbarActions>
    </Topbar>
  );
};
```

#### Topbar props

| Prop             | Type              | Description                                                                                 | Default  |
| ---------------- | ----------------- | ------------------------------------------------------------------------------------------- | -------- |
| children?        | `React.ReactNode` |                                                                                             |          |
| element?         | `string`          | Overrides the default element name to apply unique styles with the Customization Provider.s | 'TOPBAR' |
| topbarSkipLinkID | `string`          | Unique ID used to identify the Topbar element for Skip Link functionality.                  |          |

#### TopbarActions props

| Prop      | Type              | Description                                                                                   | Default          |
| --------- | ----------------- | --------------------------------------------------------------------------------------------- | ---------------- |
| children? | `React.ReactNode` | Recommended: Account Switcher, User Dialog, Product Switcher, Search Input, etc.              |                  |
| element?  | `string`          | Overrides the default element name to apply unique styles with the Customization Provider,    | 'TOPBAR_ACTIONS' |
| justify   | `"start", "end"`  | Used to control where the actions are aligned visually, to the start or end of the container. |                  |
