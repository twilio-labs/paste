## Guidelines

### About Toggle

A toggle is an interactive binary switch. It should be used in forms when an "on/off" or "yes/no" input is needed.

### Accessibility

Toggle has `role="switch"`. `aria-labelledby` is automatically set on the component, which ensures that assisstive technology devices can read the Toggle's label. When the Toggle is on, `aria-selected` is set to `"true"` (it defaults to `"false"`).

## Usage Guide

### API

#### Installation

```bash
yarn add @twilio-paste/toggle - or - yarn add @twilio-paste/core
```

#### Usage

```jsx
import {Toggle} from '@twilio-paste/core/toggle';

export const Basic = () => {
  return <Toggle>Label</Toggle>;
};
```

#### Props

| Prop             | Type                           | Description                                                                               | Default    |
