## Guidelines

### About Toggle

A Toggle is an interactive binary switch. It should be used in forms when an "on/off" or "yes/no" input is needed. Use label text to describe the purpose of the Toggle. The label text shouldn't change when the state of the Toggle changes. Help Text can also be used to provide additional guidance.

### Accessibility

Toggle has `role="switch"`. The `aria-labelledby` attribute is automatically set on the component, which ensures that assisstive technology devices can read the Toggle's label. When the Toggle is on, `aria-selected` is set to `"true"` (it defaults to `"false"`).

## Usage Guide

### API

#### Installation

```bash
yarn add @twilio-paste/toggle - or - yarn add @twilio-paste/core
```

#### Usage

```jsx
import {Toggle} from '@twilio-paste/core/toggle';

export const Basic = () => {
  return <Toggle>Label</Toggle>;
};
```

#### Props

| Prop        | Type                           | Description                                                                               | Default    |
| ----------- | ------------------------------ | ----------------------------------------------------------------------------------------- | ---------- |
| `children`  | `NonNullable<React.ReactNode>` | Pass the label text as children                                                           |            |
| `element?`  | `string`                       | Overrides the default element name to apply unique styles with the Customization Provider | `'TOGGLE'` |
| `disabled?` | `boolean`                      |                                                                                           | `false`    |
| `helpText?` | `string` or `ReactNode`        |                                                                                           | `null`     |
| `id?`       | `string`                       |                                                                                           | `null`     |
| `on?`       | `boolean`                      | Controls the state of the Toggle                                                          | `off`      |
| `required?` | `boolean`                      |                                                                                           | `false`    |
