### The new tokens page is live!

We are extremely excited to announce that we have released a new version of the tokens list page! It has a streamlined and improved design with the ability to search through token names and values, quick copy buttons for each token, and switch between JavaScript and CSS formats. ### Figma Slot components

<ResponsiveImage src={FigmaSlots} />

On July 28th, there was a big Figma release that introduced the concept of Slots. Slots are placeholder components that can be nested inside container components. They‚Äôre intentionally empty so that you can swap them out for your own custom content. It‚Äôs now easier for you to place custom content inside of existing Paste container components (e.g. Card, Modal, Disclosure, etc.) without having to detach them! It‚Äôs as easy as swapping out variants.

## üî¨ Paste by numbers

Did you know that we have data on Paste usage? We track:

- \# of repositories that import Paste packages
- Which components are used in code
- What props people use on each component

Since January 1, 2022, of the repositories using 300 or more components:

- **Console** has gone from using **18,378** components to **30,778**! This is a **67% increase** in total usage.
- **SendGrid** has gone from using **1,847** components to **5,090**! This is a **175% increase** in total usage.
